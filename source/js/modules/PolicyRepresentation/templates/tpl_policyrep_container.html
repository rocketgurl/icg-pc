Policy Representations

Policy representations are XML files and will open in a new window. Additional authentication may be required.

  • pxServer API

    XML Representation of a quote or policy (in the form of an Insurance Policy) from Insight's Quoting & Binding Platform.

  • pxCentral API

    • Default
    • v2.8
    • v2.4
    • First bound version
    • Raw
  • mxServer API

    XML Representation of a policy (in the form of an Insurance Policy) from the Insight's Policy Management Platfom

  • Agencies API

    XML representation of the sub-agency associated with this quote/policy.
